Certifique-se de que os 3 arquivos (index.html, m4.js, cozinha.js) estão na mesma pasta.

Crie uma pasta chamada images dentro dessa pasta.

Coloque suas imagens lá (floor.jpg, wall.png, earth.jpeg, etc.).

Nota: Se você não tiver todas as imagens agora, o código vai mostrar um quadrado azul ou a imagem do "earth.jpeg" se você alterar no código.

Rode o servidor local e abra o index.html.
